硬盘故障

平台

芯片：TI AM5728
SDK版本：04.02.00.09
http://software-dl.ti.com/processor-sdk-linux/esd/AM57X/04_02_00_09/index_FDS.html
内核版本：
Based on Version: 4.9.59
URL: git://git.ti.com/processor-sdk/processor-sdk-linux.git
Branch: processor-sdk-linux-04.02.00
Commit ID: a75d8e93056181d512f6c818e8627bd4554aaf92

问题：

sata接口硬盘（HDD&SSD）使用过程发生故障。
故障包括:

  • 1.只读，
  • 2.不可读写，
  • 3.可读写，但速度降到10k/s.
    以上问题对硬盘重新分区格式化后恢复正常。
